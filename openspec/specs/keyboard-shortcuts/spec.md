# Keyboard Shortcuts

## Purpose
Provide keyboard shortcuts for common application operations. TBD.

## Requirements

### Requirement: Keyboard shortcuts for common actions
The system SHALL support keyboard shortcuts for common operations.

#### Scenario: Ctrl+C copies data
- **WHEN** the user presses Ctrl+C
- **THEN** data SHALL be copied to the clipboard (same as clicking "Copiar dados")

#### Scenario: Ctrl+Shift+C copies chart
- **WHEN** the user presses Ctrl+Shift+C
- **THEN** the chart image SHALL be copied to the clipboard (same as clicking "Copiar gráfico")

#### Scenario: Ctrl+S exports PNG
- **WHEN** the user presses Ctrl+S
- **THEN** the export PNG dialog SHALL open (same as clicking "Exportar PNG")

#### Scenario: F5 fetches data
- **WHEN** the user presses F5
- **THEN** data SHALL be fetched for the currently entered date

#### Scenario: Ctrl+E toggles evolution
- **WHEN** the user presses Ctrl+E
- **THEN** the evolution checkbox SHALL be toggled

#### Scenario: Ctrl+L toggles analysis sidebar
- **WHEN** the user presses Ctrl+L
- **THEN** the analysis sidebar SHALL be toggled
